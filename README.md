# 📘 Git School

Este repositorio contiene una guía introductoria orientada a comprender y utilizar **Git** y **GitHub** como herramientas fundamentales para el control de versiones y el trabajo colaborativo.

## 🎯 Objetivo

Proporcionar una base clara y práctica para:

- Entender los conceptos esenciales de Git
- Familiarizarse con el uso de repositorios
- Comprender la lógica de trabajo colaborativo en GitHub

## 📂 Contenido del repositorio

El material está organizado de forma progresiva, abarcando:

- Conceptos clave de control de versiones
- Uso de repositorios locales y remotos
- Manejo de ramas
- Integración de cambios
- Buenas prácticas en proyectos

## 🧩 Enfoque

La guía está diseñada bajo un enfoque práctico, priorizando:

- Claridad en los conceptos
- Aplicación directa en proyectos
- Estructura simple y organizada

## 👥 Público objetivo

- Principiantes en programación
- Estudiantes que buscan mejorar su flujo de trabajo
- Cualquier persona interesada en el desarrollo colaborativo

## 📌 Notas

Este repositorio actúa como punto de partida.
Se recomienda complementar el aprendizaje con práctica constante y proyectos reales.

---

💡 *Dominar Git no es memorizar comandos, sino entender cómo evoluciona tu proyecto a lo largo del tiempo.*
